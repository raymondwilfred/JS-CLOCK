# JS-CLOCK

A lightweight, dynamic clock built to practice JavaScript Date objects and asynchronous UI updates.

Key Features:

12-Hour Formatting: Implemented logic to convert standard 24-hour time to a user-friendly 12-hour format with AM/PM toggles.

Real-Time DOM Updates: Utilized setInterval() to ensure the UI refreshes every second without page reloads.

Zero-Padding: Wrote logic to ensure single-digit minutes/seconds are formatted correctly (e.g., 1:05:02 instead of 1:5:2).
